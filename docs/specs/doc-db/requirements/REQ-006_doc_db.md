# REQ-006: doc-db plugin 要件定義書

## 概要

doc-db は、プロジェクトのドキュメントを **見出し単位 chunk Embedding + Lexical Hybrid + LLM Rerank** で検索する plugin である。doc-advisor の ToC 方式（高品質だが AI 介在で遅い）と並行運用し、ユーザーが用途に応じて使い分ける。

doc-advisor 本体には影響を与えない独立した恒久 plugin として育てる。

## 前提条件

| ID     | 要件                                                                                              |
| ------ | ------------------------------------------------------------------------------------------------- |
| PRE-01 | `.doc_structure.yaml` がプロジェクトルートに存在する（doc-advisor と共有）                        |
| PRE-02 | Embedding API および Chat Completion API（モデル選定は設計書）の API key が環境変数で利用可能である（参考実装: OpenAI `text-embedding-3-large` / `gpt-4o-mini`） |
| PRE-03 | doc-db は doc-advisor 本体の挙動を変更しない（並行運用）                                          |

## 受け入れ条件

| ID    | 条件                                                                                  |
| ----- | ------------------------------------------------------------------------------------- |
| AC-01 | doc-db plugin が doc-advisor 本体と独立して動作する                                   |
| AC-02 | 同一ゴールデンセット（`meta/test_docs/`）で ToC 方式・現行 Embedding 方式と比較できる |
| AC-03 | NFR-004 TBD-009 で定義する recall 同等基準を満たす                                    |

## 用語定義

| 用語             | 定義                                                                                                  |
| ---------------- | ----------------------------------------------------------------------------------------------------- |
| chunk            | 検索の最小単位。Markdown の見出し（`#` 以下の section）を境界として 1 chunk を構成する                |
| Multi-vector     | 1 文書を複数のベクトルで表現する方式。chunk ごと、または field（title / body 等）ごとにベクトルを持つ |
| Lexical Hybrid   | Embedding による意味検索と Lexical Search（語彙一致ベースの検索）を組み合わせ、両スコアを統合する手法（具体的なスコア式・統合方式は設計書で確定） |
| LLM Rerank       | 取得した候補 chunk を LLM に渡し、関連度順に並び替える後処理（モデル選定は設計書で確定）              |
| ゴールデンセット | 期待する検索結果が定義された評価用クエリ集合（`meta/test_docs/`）                                     |
| false negative   | 関連あるはずの文書が検索結果に含まれない見落とし                                                      |
