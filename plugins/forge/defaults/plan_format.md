# 計画書 デフォルトフォーマット

作成原則: `${CLAUDE_PLUGIN_ROOT}/defaults/plan_principles.md`

| - | -------- | ---------- | -------- |

## 2. 設計トレーサビリティマトリクス

| 設計ID   | タイトル   | 要件ID   | タスクID   |
| -------- | ---------- | -------- | ---------- |

## 3. タスク一覧

| ✓ | 優先度 | タスクID | タイトル   | やるべき内容           | 設計ID   | 依存関係 | グループID | ビルド確認 | 必読                |
| - | ------ | -------- | ---------- | ---------------------- | -------- | -------- | ---------- | ---------- | ------------------- |
| ☐ | {1-99} | TASK-001 | {タイトル} | ・{項目1}<br>・{項目2} | {設計ID} | -        | -          | タスクごと | {ファイルパス or -} |

- 高（70〜99）: コアビジネスロジック・共通基盤・ブロッカー解消
- 中（40〜69）: 主要機能
- 低（1〜39）: UI・補助機能
